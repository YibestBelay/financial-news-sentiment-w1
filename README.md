# Financial News Sentiment – Week 1

This project performs Exploratory Data Analysis (EDA) on financial news headlines as part of Week-1 of the 10Academy challenge.

## How to Run

1. Activate your virtual environment:
   .venv\Scripts\Activate.ps1

2. Install dependencies:
   pip install -r requirements.txt

3. Place your dataset in the project root and name it:
   news.csv

4. Run the EDA script:
   python src/eda_news.py --input news.csv --outdir outputs
